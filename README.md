# 🐍 Modern Snake Game

A sleek, responsive, and high-performance Snake game built with **React**, **TypeScript**, and **HTML5 Canvas**. This project features a dual-theme system, haptic feedback, and dynamic difficulty scaling.

🚀 **Live Demo:** [https://farmanullah1.github.io/Snake-Game/](https://farmanullah1.github.io/Snake-Game/)

---

## ✨ Key Features

* **🎮 Advanced Gameplay:** Three difficulty modes (Chill, Normal, Turbo) with dynamic speed scaling as your score increases.
* **🌓 Dual Theme System:** * **Light Mode:** A soft, modern gradient aesthetic.
    * **Dark Mode:** A neon-cyber gaming vibe with glow effects.
* **📱 Fully Responsive:** Edge-to-edge mobile optimization with custom on-screen D-Pad and swipe gesture support.
* **📳 Haptic Feedback:** Integrated vibration API for a tactile experience on mobile devices.
* **🏆 Persistence:** LocalStorage integration to track and save high scores across different difficulty levels.
* **⚡ Built with Vite:** Optimized for ultra-fast performance and smooth 60FPS canvas rendering.

---

## 🛠️ Technical Stack

* **Framework:** [React 19](https://react.dev/)
* **Language:** [TypeScript](https://www.typescriptlang.org/)
* **Rendering:** HTML5 Canvas API
* **Build Tool:** [Vite](https://vitejs.dev/)
* **Deployment:** GitHub Pages
